Installation
============

Requirements
------------

* Python 3.12 or higher

Install from PyPI
-----------------

.. code-block:: bash

   pip install tacular

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/tacular-omics/tacular.git
   cd tacular
   pip install -e .

Using uv (recommended for development)
--------------------------------------

.. code-block:: bash

   git clone https://github.com/tacular-omics/tacular.git
   cd tacular
   uv sync
